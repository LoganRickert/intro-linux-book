Chapter 5 Networking Tools

We will deal with the lower layers of the networking stack. The transport layer, network layer, data link layer, and physical layer are all needed in order to use the internet. The Transport Layer splits up longer messages into multiple, smaller messages. The Network Layer contains information about where the traffic should go. The Data Link Layer

is the infrustructure to figure out how to handle the traffic that is coming in and out. The Physical Layer are the physical connections, such as ethernet cables.
