Chapter 4 Analysing Binary Files

strings

Manual #4.1
STRINGS
Command
Description

Finds the printable strings in an object or binary file.

ltrace

Manual #4.2
LTRACE
Command
Description

Runs the specified program until it exits. It intercepts and records the dynamic library calls which are called by the executed process and the signals which are received by that process.

strace

Manual #4.3
STRACE
Command
Description

Runs the specified program until it exits. It intercepts and records the system calls which are called by a process and the signals which are received by a process.

File

Manual #4.4
FILE
Command
Description

Prints out information about the file.
