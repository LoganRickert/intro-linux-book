Chapter 1 Introduction To Linux

Linux is a free, open-source operating system. The design of the operating system is based on Unix, an older operating system. A lot of the work that happens on Linux is done from inside of a text-based application called a terminal. It can be challenging to get use to and work with the terminal at first but with a little bit of practice it becomes blazing fast compared to most graphical user interfaces. It's important to be fimilar with the terminal because almost all server-based applications and systems do not have a graphical user interface. Files have to be moved around, applications have to be installed, and much more all have to happen from within the terminal.

The most important command in this book to know is the man command.

Manual #2.1
LS
Command
Description

List information about the FILEs (the current directory by default).

Common Flags
  • l - List extended information about the file
  • a - List all files, including hidden files (files starting with a .
  • h - Show the size of the files in a human readable format.
  • t - Sort by time last modified with the newest file being first.
